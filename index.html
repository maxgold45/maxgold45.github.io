Work:

PivotalPath: Software Engineering Internship

I developed an app which we called the PivotalPath Score Card. Each component is created dynamically so that the charts are interactive and change based upon the parameters and settings chosen. Below you can see two of the many components I developed. These were developed using Plotly Dash and data was cleaned using the Python pandas data analysis library.

The first component is the Return Table. This is used to see how a fund performs over time. You can see how it performs monthly, quarterly, year to date. You can also see the drawdown, which is the peak to trough decline.

Display Return Table Image

The second component is the Z Score graph. With this component, you can see how a fund performs compared to its peer group. A peer group is a group of funds that are similar to the selected fund. For example, if Microsoft were a hedge fund, then companies like Apple and Google would be in its peer group. The fund in this graph only outperformed its peer group in 2014. Every other year, it underperformed.
